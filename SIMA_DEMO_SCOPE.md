# SIMA_DEMO_SCOPE

SIMA is a demo frontend for an AI meeting assistant system.

## Goal of the demo
Demonstrate a clean and futuristic interface for:
1. Logging in
2. Starting a meeting
3. Showing a live listening/writing assistant experience
4. Displaying the resulting transcription

## Screen 1: Login
- Username input
- Password input
- Login button
- Premium centered layout
- Animated blurred purple-black background

## Screen 2: Home
- Minimal dashboard
- Main action: start a new meeting
- Secondary cards: recent session, system status, or quick access
- Clean and not overloaded

## Screen 3: Meeting
- Most important screen
- Same animated premium background
- Central round button to start/stop meeting
- Meeting status
- Timer
- Simple audio visualization
- Area for animated assistant
- Assistant states:
  - listening
  - writing
- The assistant should feel friendly, expressive, and memorable

## Screen 4: Transcript
- Clean transcription result page
- Header with meeting info
- Main transcript panel
- Summary/info side panel
- Action buttons such as copy, export, start new meeting

## UI personality
- Futuristic but institutional
- Friendly but not childish
- Memorable visual identity
- Demo-ready and presentation-ready
